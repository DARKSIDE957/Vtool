# Vtool Avatar Auto-Fixer Pro

Welcome to the ultimate "A to Z" VRChat Avatar Fixer. This tool automatically diagnoses and fixes the most common Unity errors that prevent avatars from uploading to VRChat or cause issues in-game.

## Features
- **Diagnostics Dashboard:** Instantly view polycount, material slots, and skinned mesh metrics.
- **1-Click Master Fix:** Runs a suite of safe, essential fixes instantly.
- **Missing Scripts & Materials Cleaner:** Strips out broken references to prevent build errors.
- **VRChat Auto-Setup:** Automatically aligns your View Position and sets up Visemes for Lip Sync.
- **Quest / Android Conversion:** One-click conversion of all materials to the highly optimized `VRChat/Mobile/Toon Lit` shader required for Quest uploads.
- **Optimization Tools:** Safely unpack prefabs and clean up empty GameObjects from bloated hierarchies.

## Installation via VRChat Creator Companion (VCC)

To use this tool, you can easily install it into your Unity projects using the VCC Local Packages feature.

**The Repository URL:**
`https://github.com/DARKSIDE957/Vtool`

### Step-by-Step Installation:
1. Download or clone this repository to a folder on your computer. 
2. Open the **VRChat Creator Companion**.
3. Go to the **Projects** tab on the left.
4. Click **Manage Project** on the avatar project where you want to install Vtool.
5. In the top right corner of the package list, click the small **dropdown arrow** next to the `+` button.
6. Select **Add Local Package**.
7. In the file explorer, navigate to the folder where you saved this repository and select it.
8. The package will instantly be added to your project!

## Usage
Once installed, open your Unity project. You will find the tool at the top of the screen:
**`VRChat -> Avatar Auto-Fixer Pro`**

Drag your avatar's root GameObject into the tool to unlock all the fixing tabs.
